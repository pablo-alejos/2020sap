Etiquetas

Las etiquetas sirven para clasificar tus publicaciones de productos académicos, hacerlos mas fácil de encontrar y mejorar las métricas que genera el sistema.

Si crees que hace falta una etiqueta puedes agregarla mientras cumpla con los siguientes requisitos:

- Corresponda o represente a una linea de generacion y aplicacion del conocimiento(LGAC).

- No exista una etiqueta representando la misma LGAC.

El administrador del sistema siempre puede eliminar o modificar cualquier etiqueta.
